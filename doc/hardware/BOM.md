# Hardware components used to build the MultiChannelAmp

| Count | Component | Notes |
|---|---|---|
| 1 | Raspberry Pi5 | I decided to use a Pi5 to prevent any issues with USB, CPU or networking performance. |

* Raspberry pi5
* Waveshare Power Over Ethernet HAT G
* 3x SURE KAB9 USB sound card
* Meanwell HRPG-600-24
* DC 5v-36v 400W FET Trigger Switch Drive Module Board (to drive the fan using PWM signal)
* Sunon MagLev MEC038V2-000U-A99 fan
* Hailege PC817 8 Channel Optokoppler Module
